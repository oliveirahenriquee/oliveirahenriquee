 Stack Tecnológica

- Front-end
*Core: HTML5 Semântico, CSS3 (Flexbox/Grid, Design Responsivo, Glassmorphism)
*Interatividade: JavaScript moderno (ES6+)

- Back-end & Banco de Dados
*Runtime & Frameworks: Node.js (ES Modules), Express.js
*Bancos de Dados: MySQL (Pool de conexões, Queries Otimizadas, Integração em Nuvem)
*Ferramentas & APIs: `whatsapp-web.js`, Puppeteer, JWT (Autenticação Segura), Bcrypt, Helmet, Rate Limiting

- Ferramentas & Infraestrutura
*Versionamento: Git & GitHub (Clean Commits e Organização Arquitetural)
*Hospedagem & Cloud: Render (Persistência de Dados e Volumes), Aiven (Cloud Databases)

------------------------------------------------------------------------------------------


- Projetos em Destaque

- Controle Financeiro Inteligente & WhatsApp Bot(https://github.com/oliveirahenriquee/sistema-de-gastos)
Um ecossistema completo que une uma API robusta em Node.js com um bot automatizado no WhatsApp. O usuário faz lançamentos e consultas via chat, os dados são consolidados na nuvem (Aiven) e alimentam um dashboard web interativo com gráficos em tempo real.
*Destaques técnicos: Migração para ES Modules, Clean Code dividido em módulos de segurança/rotas/conexões, persistência de sessão Chromium via volumes no Render e proteção contra SQL Injection e ataques DoS.
